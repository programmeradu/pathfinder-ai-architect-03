## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ or Bun
- Modern web browser
- Internet connection for API calls

### Installation & Development

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd pathfinder-ai
   ```

2. **Install dependencies**
   ```bash
   # Using npm
   npm install
   
   # Using bun (recommended)
   bun install
   ```

3. **Start development server**
   ```bash
   # Using npm
   npm run dev
   
   # Using bun
   bun dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Building for Production

```bash
# Build the application
npm run build

# Preview the build
npm run preview
```
